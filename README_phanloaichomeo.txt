1. Dữ lịiệu 
- Nguồn thu thập dữ liệu: kaggle
- Sử dụng file kaggle.json để tải dữ liệu.
- Các bộ dữ liệu được sử dụng:

  1. Cats vs Dogs Training8000Test2000
     Link: https://www.kaggle.com/datasets/dhirensk/cats-vs-dogs-training8000test2000

  2. Microsoft Cats vs Dogs Dataset
     Link: https://www.kaggle.com/datasets/shaunthesheep/microsoft-catsvsdogs-dataset
     Sử dụng để bổ sung dữ liệu cho bài toán phân loại chó hoặc mèo.

  3. The Oxford-IIIT Pet Dataset
     Link: https://www.kaggle.com/datasets/tanlikesmath/the-oxfordiiit-pet-dataset
     Sử dụng cho bài toán phân loại giống chó và mèo.
